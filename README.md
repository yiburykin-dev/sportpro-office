# SPORTPRO Office

Локальное веб-приложение для учета рекламы, воронки продаж и скриптов общения с клиентами SPORTPRO.

## Что внутри

- Раздел `Реклама`: дашборд, контент, воронка.
- Раздел `Продажи`: интерактивный скрипт, возражения, анкета клиента.
- Темы оформления.
- Экспорт CSV.
- Облачная синхронизация через Supabase.

## Запуск

Откройте `index.html` в браузере.

## GitHub Pages

Для публикации как сайта используйте `index.html` в корне репозитория и включите GitHub Pages для ветки `main`.

## Supabase

Для облачной синхронизации выполните SQL из `supabase-setup.sql`, затем в приложении откройте `Реклама -> Облако` и укажите Project URL, publishable key и workspace.
